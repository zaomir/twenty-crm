## 🎯 Задача

Встроить справку (Help) в меню Settings приложения controlcenter.me так, чтобы пользователи могли:
- Открыть Settings
- Нажать Help
- Увидеть полную справку прямо в приложении

---

## 📋 Требования

### Функциональные требования

✅ **Settings → Help**
- Добавить пункт меню "Help" в Settings
- При клике открывается справка

✅ **Содержимое справки**
- Использовать текст из `IN-APP-HELP-PAGE.md`
- Отформатировать в HTML/React компонент
- Сохранить структуру (заголовки, подразделы, таблицы, примеры)

✅ **Навигация**
- Содержание с якорями (Table of Contents → быстрый переход)
- Поиск по справке (Ctrl+F должен работать)
- Кнопка "Back" или закрытие

✅ **Мобильная версия**
- Справка должна открываться на мобильных
- Читаемость на экране < 600px

### Технические требования

✅ **React Component**
- Создать компонент `HelpPage.jsx` (или `.tsx`)
- Используйте Markdown parser если нужно (markdown-to-jsx, react-markdown)
- Или просто HTML структура

✅ **Стилизация**
- Используйте существующие стили приложения
- Цветовая схема как в остальном UI
- Читаемый шрифт (16px+)

✅ **Доступность**
- Правильные заголовки (h1, h2, h3)
- Alt текст для изображений (если будут)
- Контрастность текста

## 📂 Файлы которые нужно изменить

### 1. Создать новый компонент

```
src/components/settings/HelpPage.jsx (или .tsx)
```

Структура:
```jsx
export function HelpPage() {
  return (
    <div className="help-page">
      <nav className="help-toc">
        {/* Table of Contents с якорями */}
      </nav>
      <article className="help-content">
        {/* Контент из IN-APP-HELP-PAGE.md */}
      </article>
    </div>
  );
}
```

### 2. Добавить в Settings меню

```
src/components/settings/SettingsMenu.jsx (или аналог)
```

Найти где выводится меню Settings, добавить:
```jsx
<MenuItem
  label="Help"
  icon={<QuestionMarkIcon />}
  onClick={() => navigate('/settings/help')}
/>
```

### 3. Добавить маршрут

```
src/routes/settingsRoutes.jsx (или аналог)
```

Добавить:
```jsx
{
  path: '/settings/help',
  element: <HelpPage />
}
```

## 📞 Вопросы при реализации?

**Q: Где находится Settings компонент?**  
A: Обычно в `src/pages/Settings/` или `src/components/Settings/`

**Q: Какой парсер Markdown использовать?**  
A: `react-markdown` или просто HTML структура (проще всего)

**Q: Как добавить якоря для Table of Contents?**  
A: Используйте `id` на заголовках и `href="#id"` в навигации

**Q: Как реализовать поиск?**  
A: Браузерный Ctrl+F работает автоматически. Если нужен свой поиск — используйте state + фильтрацию

**Q: На мобильном как скрыть TOC?**  
A: `display: none` на экранах < 768px, и toggle кнопка для показа

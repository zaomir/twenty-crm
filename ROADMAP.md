## 🌐 ФАЗА 2: ИНТЕГРАЦИИ

### 2.1 Google Calendar

**Что нужно:**
- [ ] Google Cloud Project (создать в console.cloud.google.com)
- [ ] Google Calendar API enabled
- [ ] OAuth 2.0 credentials (Client ID + Secret)

**Что делать:**
1. Задачи в Twenty CRM синхронизируются с Google Calendar
2. События из Google Calendar показываются в Twenty CRM
3. Двусторонняя синхронизация (изменение в одном = изменение в другом)

**Документация:**
- Инструкция для пользователей как подключить
- Инструкция для администратора как настроить

---

### 2.2 Todoist

**Что нужно:**
- [ ] Todoist API token (из Todoist Settings)
- [ ] Todoist Project ID (какой проект синхронизировать)

**Что делать:**
1. Задачи из Twenty CRM экспортируются в Todoist
2. Задачи из Todoist импортируются в Twenty CRM
3. При изменении статуса в одном — меняется в другом

**Документация:**
- Инструкция как подключить Todoist
- Какие данные синхронизируются

---

### 2.3 Google Contacts

**Что нужно:**
- [ ] Google Contacts API enabled
- [ ] OAuth 2.0 credentials

**Что делать:**
1. Контакты из Twenty CRM можно экспортировать в Google Contacts
2. Контакты из Google Contacts можно импортировать в Twenty CRM
3. Синхронизация по email (один контакт = один email)

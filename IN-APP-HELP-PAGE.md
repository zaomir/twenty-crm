# 📖 Памятка пользователю controlcenter.me

## 🎯 Что это?

**controlcenter.me** — это **CRM система** для управления:
- 👥 **Контактами** (люди, с которыми вы общаетесь)
- 💼 **Компаниями** (организации)
- 🤝 **Сделками** (возможности продажи)
- 📅 **Задачами** (что нужно сделать)

---

## 🚀 Быстрый старт (5 минут)

### Шаг 1: Добавить первый контакт
```
👥 Contacts → + Add Contact
├─ Имя
├─ Email
├─ Телефон
├─ Компания
└─ Должность
```

### Шаг 2: Добавить компанию
```
💼 Companies → + Add Company
├─ Название
├─ Сайт
├─ Телефон
└─ Адрес
```

### Шаг 3: Добавить сделку
```
🤝 Deals → + Add Deal
├─ Название
├─ Компания
├─ Контакт
├─ Сумма
├─ Статус
└─ Дата закрытия
```

---

## 📚 Основные разделы

### 👥 Контакты
**Люди, с которыми вы общаетесь**
- Сохраняйте информацию о каждом человеке
- Привязывайте контакт к компании
- Добавляйте примечания
- Создавайте задачи для контактов

### 💼 Компании
**Организации, с которыми вы работаете**
- Видите всех контактов из компании
- Отслеживаете историю взаимодействия
- Смотрите все открытые сделки

### 🤝 Сделки (Deals)
**Возможности продажи или сотрудничества**

**Статусы:**
- 🆕 New / Interest
- 💬 Negotiation (переговоры)
- ✅ Ready for Closure (готово подписать)
- ✅✅ Won (выиграна)
- ❌ Lost (потеряна)

### 📅 Задачи (Tasks)
**Действия, которые нужно выполнить**
- Позвонить
- Отправить документы
- Подготовить договор
- Проверить платёж

### 📊 Pipeline (Воронка продаж)
**Вся статистика на один взгляд**
- Сколько сделок на каждом этапе
- Общий потенциальный доход
- Прогноз по статусам

---

## 💡 Лучшие практики

### ✅ ДЕЛАЙТЕ:
- ✅ Добавляйте контакты **сразу** после встречи
- ✅ **Обновляйте статус сделки** после каждого контакта
- ✅ **Создавайте задачи** для следующих действий
- ✅ **Добавляйте примечания** к контактам и сделкам
- ✅ Проверяйте **Pipeline** еженедельно

### ❌ НЕ ДЕЛАЙТЕ:
- ❌ Не забывайте добавлять контакты
- ❌ Не оставляйте сделки с устаревшим статусом
- ❌ Не создавайте название "Клиент 1" или "Сделка A"
- ❌ Не забывайте про задачи

## 🔍 Быстрые действия

### Поиск контакта
```
Contacts → Поле поиска → Введите имя/email → Enter
```

### Изменить статус сделки
```
Deals → Выберите сделку → Нажмите статус → Выберите новый
```

### Добавить примечание
```
Откройте контакт → Comments → Напишите примечание → Save
```

### Назначить задачу коллеге
```
Tasks → Выберите задачу → Assigned to → Выберите сотрудника
```

---

## 👥 Работа в команде

### Пригласить коллегу
```
Settings → Users → Add User → Email коллеги
```

### Назначить задачу
```
Задача → Assigned to → Выберите сотрудника
```

### Видеть что делают другие
```
Раздел → Фильтр по пользователю или просмотрите недавние изменения
```

## 🆘 Частые вопросы

**Q: Как экспортировать данные?**  
A: Раздел → меню ⋮ → Export → CSV/Excel

**Q: Как удалить контакт?**  
A: Откройте контакт → меню ⋮ → Delete (осторожно — не восстанавливается!)

**Q: Что если потеряны данные?**  
A: Обратитесь к администратору — есть резервные копии

**Q: Как изменить валюту?**  
A: Settings → Workspace → Currency

**Q: Как пригласить коллег?**  
A: Settings → Users → Add User

**Q: Работает ли с мобильного?**  
A: Да! Откройте в браузере мобильного.

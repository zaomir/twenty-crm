📥 КАК ИСПОЛЬЗОВАТЬ

Вариант 1: Клонировать в Cursor Terminal
──────────────────────────────────────────
1. Откройте Cursor
2. Ctrl+` (открыть Terminal)
3. Выполните:

   git clone https://github.com/zaomir/twenty-crm.git
   cd twenty-crm

4. Все файлы готовы к использованию!

Вариант 2: Откройте в Cursor через GitHub
───────────────────────────────────────────
1. File → Open from Repository
2. Введите: https://github.com/zaomir/twenty-crm
3. Выберите папку для клонирования

Вариант 3: Скачайте ZIP
──────────────────────
1. Откройте: https://github.com/zaomir/twenty-crm
2. Code → Download ZIP
3. Распакуйте

════════════════════════════════════════════════════════════════════════════

🤖 ИСПОЛЬЗУЙТЕ С CURSOR

После клонирования:

1. Cmd+K (Mac) / Ctrl+K (Windows)
2. Откройте: CURSOR-БЫСТРЫЙ-СТАРТ.md
3. Скопируйте "Prompt 1: Развёртывание"
4. Вставьте в Chat
5. ENTER
6. Ждите 15 минут

ГОТОВО! Twenty CRM развёрнут! 🎉

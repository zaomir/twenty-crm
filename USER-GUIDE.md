# 👤 USER-GUIDE.md — Инструкция для пользователей controlcenter.me

**Добро пожаловать в controlcenter.me!** 🎉

Это руководство для всех, кто просто хочет **использовать** систему управления контактами и сделками.

---

## 🎯 Что такое controlcenter.me?

**controlcenter.me** — это **бесплатная CRM система** (Customer Relationship Management), которая помогает:

✅ **Управлять контактами** — сохранять информацию о клиентах, партнерах, поставщиках  
✅ **Отслеживать сделки** — вести переговоры, отслеживать статус, прогнозировать доход  
✅ **Организовать работу** — распределять задачи, устанавливать приоритеты, видеть что кто делает  
✅ **Анализировать данные** — понимать кто ваши лучшие клиенты, какие сделки наиболее прибыльны  

**Аналоги:** Salesforce, HubSpot, Pipedrive (но бесплатно и на вашем сервере)

## 🚀 Как начать?

### Шаг 1: Открыть в браузере

```
https://controlcenter.me
```

### Шаг 2: Создать учётную запись

При первом входе вы создадите администратора:

- **Email:** ваш рабочий email
- **Пароль:** надёжный пароль
- **Организация:** название вашей компании

### Шаг 3: Первый вход

После входа вы увидите главное меню:

```
👥 Contacts (Контакты)
💼 Companies (Компании/Организации)
🤝 Deals (Сделки)
📅 Tasks (Задачи)
📊 Pipeline (Воронка продаж)
⚙️ Settings (Настройки)
```

## 📚 Основные концепции

### 👥 Контакты

**Что это?** Люди, с которыми вы общаетесь.

**Примеры:**
- Иван Петров — директор компании "ООО Рога и Копыта"
- Мария Сидорова — менеджер по закупкам компании "ИП Васильев"

**Как добавить?**
1. Откройте раздел **Contacts**
2. Нажмите **+ Add Contact**
3. Заполните:
   - Имя
   - Email
   - Телефон
   - Компания (если есть)
   - Должность
   - Другая информация

**Зачем это нужно?**
- Иметь в одном месте всех, с кем вы общаетесь
- Не потерять информацию о клиентах
- Быстро найти контакт по имени, email, телефону

---

### 💼 Компании (Organizations)

**Что это?** Организации, с которыми вы работаете.

**Примеры:**
- ООО Рога и Копыта
- ИП Васильев
- АО Рыбный промысел

**Как добавить?**
1. Откройте раздел **Companies**
2. Нажмите **+ Add Company**
3. Заполните:
   - Название компании
   - Сайт
   - Телефон
   - Адрес
   - Другая информация

**Связь с контактами:**
- У каждого контакта есть ссылка на компанию
- На странице компании видны все контакты из этой компании

**Зачем это нужно?**
- Видеть всю структуру работы с компанией
- Знать кто в какой компании работает
- Отслеживать историю работы с каждой организацией

---

### 🤝 Сделки (Deals)

**Что это?** Возможности сделать какую-то продажу или сотрудничество.

**Примеры:**
- Поставка 1000 тонн рыбы компании "Рога и Копыта" — сумма 5 млн рублей
- Консультация для компании "Васильев" по оптимизации закупок — сумма 500 тыс рублей

**Как добавить?**
1. Откройте раздел **Deals**
2. Нажмите **+ Add Deal**
3. Заполните:
   - Название сделки
   - Компания
   - Контакт
   - Сумма (в рублях/долларах/евро)
   - Статус (новая, переговоры, готово к закрытию, выиграна, потеряна)
   - Дата закрытия (когда вы планируете завершить)
   - Описание

**Статусы сделок (воронка продаж):**
```
1. New / Interest        → Только узнали о возможности
2. Negotiation          → Активные переговоры
3. Ready for Closure    → Почти готово подписать
4. Won                  → ✅ Сделка закрыта успешно
5. Lost                 → ❌ Сделка не состоялась
```

**Зачем это нужно?**
- Видеть сколько денег вы можете заработать
- Понимать на каком этапе находится каждая сделка
- Прогнозировать доход на месяц/квартал/год
- Не забыть о сделках и вовремя их закрыть

---

### 📅 Задачи (Tasks)

**Что это?** Действия, которые нужно выполнить.

**Примеры:**
- Позвонить Ивану Петрову завтра в 10:00
- Отправить коммерческое предложение компании "Рога и Копыта"
- Подготовить договор для сделки с "Васильевым"

**Как добавить?**
1. Откройте раздел **Tasks**
2. Нажмите **+ Add Task**
3. Заполните:
   - Название задачи
   - Контакт / Сделка (если связано)
   - Срок выполнения
   - Приоритет (Low, Medium, High)
   - Описание

**Зачем это нужно?**
- Не забыть что нужно сделать
- Видеть ваши приоритеты
- Отслеживать что выполнено, что нет

## 📊 Как работать каждый день?

### Утром (9:00)

```
1. Откройте controlcenter.me
2. Посмотрите раздел "Tasks" → 
   - Задачи с высоким приоритетом
   - Сроки выполнения которых наступили сегодня
3. Выполняйте задачи в течение дня
```

### Во время переговоров

```
Во время звонка с клиентом:
1. Откройте его контакт в controlcenter.me
2. Посмотрите историю переговоров
3. Смотрите открытые сделки с этим клиентом
4. Сразу добавляйте задачи ("Позвонить завтра", "Отправить счет")
```

### Вечером (17:00)

```
1. Откройте раздел "Deals"
2. Обновите статусы сделок (если что-то изменилось)
3. Добавьте задачи на завтра
4. Посмотрите какие сделки готовы к закрытию
```

### Еженедельно (каждый понедельник)

```
1. Откройте раздел "Pipeline" (воронка продаж)
2. Посмотрите общую картину:
   - Сколько сделок на каждом этапе
   - Какой общий размер потенциального дохода
3. Обновите прогноз на неделю/месяц
```

### Ежемесячно (1-е число)

```
1. Откройте раздел "Analytics" (если есть)
2. Посмотрите статистику:
   - Сколько новых контактов добавили
   - Сколько сделок закрыли
   - Сколько дохода получили
3. Проанализируйте: что работает, что нет
```

## 💡 Лучшие практики

### ✅ ДЕЛАЙТЕ так:

✅ **Добавляйте контакты сразу же** после встречи или звонка  
✅ **Описывайте каждую сделку подробно** — когда вы на нее посмотрите через месяц, поймете что происходило  
✅ **Обновляйте статус сделки** после каждого контакта с клиентом  
✅ **Создавайте задачи** для следующих действий ("позвонить", "отправить KP", "подписать договор")  
✅ **Присваивайте задачи** разным сотрудникам (если вас несколько)  
✅ **Ставьте приоритеты** — что срочное, что может подождать  
✅ **Добавляйте примечания** к контактам и сделкам о том как их лучше подходить  

### ❌ НЕ ДЕЛАЙТЕ так:

❌ Не забывайте добавлять контакты (запомнить в голове — рискованно)  
❌ Не оставляйте сделки с старым статусом (если переговоры завершены, обновите)  
❌ Не добавляйте названия вроде "Клиент 1" или "Сделка A" (непонятно что это)  
❌ Не забывайте о контактах (в controlcenter.me видны задачи с напоминаниями)

## 🆘 Частые вопросы

**Вопрос:** Как пригласить коллег в систему?  
**Ответ:** Settings → Users → Add User → введите email коллеги. Он получит приглашение.

**Вопрос:** Можно ли экспортировать данные?  
**Ответ:** Да, обычно есть кнопка Export в каждом разделе (CSV, Excel).

**Вопрос:** Как удалить контакт/сделку?  
**Ответ:** Откройте контакт/сделку → меню (⋮) → Delete. Осторожно — отменить нельзя!

**Вопрос:** Что если я потерял данные?  
**Ответ:** Обратитесь к администратору — система делает резервные копии еженедельно.

**Вопрос:** Как изменить валюту (рубли/доллары)?  
**Ответ:** Settings → Workspace → Currency. Выберите нужную валюту.

**Вопрос:** Можно ли работать с мобильного?  
**Ответ:** Да! controlcenter.me открывается на мобильном браузере, но пока нет мобильного приложения.

## 🚀 БЫСТРЫЙ СТАРТ (5 минут)

### 1. Клонировать репозиторий

```bash
git clone https://github.com/zaomir/twenty-crm.git
cd twenty-crm
```

### 2. Читать в этом порядке

```
📄 CURSOR.md                    ← НАЧНИТЕ ОТСЮДА
   ↓ (откройте этот файл)
   └─→ Промпт для Cursor Chat
```

### 3. Скопировать промпт в Cursor

- Откройте файл `CURSOR.md`
- Найдите раздел **"Промпт для чата Cursor"**
- Скопируйте весь текст промпта
- Откройте Cursor Chat (Cmd+K на Mac, Ctrl+K на Windows/Linux)
- Вставьте промпт (Cmd+V / Ctrl+V)
- Нажмите ENTER

### 4. Следовать инструкциям

Cursor будет направлять вас через все шаги.

## 💡 Главные правила

### ✅ ВСЕГДА помните:

```bash
# 1. Для обновления конфига (БЕЗ -v):
docker compose down
docker compose up -d

# 2. ТОЛЬКО для полного reset (С -v удаляет данные!):
docker compose down -v
docker compose up -d

# 3. .env копируется автоматически:
# Просто создайте .env из .env.example
# docker compose прочитает его автоматически

# 4. Бэкапы еженедельно:
docker exec controlcenter-postgres pg_dump \
  -U controlcenter_user controlcenter_prod > backup.sql

# 5. Ротация пароля каждые 90 дней:
# openssl rand -hex 32  # для нового пароля
```

### ❌ НИКОГДА не делайте:

- ❌ Не коммитьте `.env.production` в git
- ❌ Не публикуйте пароли в markdown
- ❌ Не используйте `-v` флаг без явного намерения
- ❌ Не пропускайте бэкапы
- ❌ Не забывайте про ротацию паролей

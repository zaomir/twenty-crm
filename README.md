## 🆘 NEED HELP?

### Quick troubleshooting

**Problem:** "Can't connect to crm.grainee.com"
```bash
# Check if containers are running
ssh root@213.155.28.121 "docker ps | grep twenty"

# Check nginx
ssh root@213.155.28.121 "sudo systemctl status nginx"

# Check DNS
nslookup crm.grainee.com
```

**Problem:** "GRAINEE stopped working"
- GRAINEE is NOT affected (different port)
- Check GRAINEE logs separately
- It's an isolated issue

**Problem:** "Google Sheets sync not working"
- See Phase 2 in `TWENTY_CRM_SETUP_COMPLETE.md`
- Check Apps Script logs in Google Sheets
- Verify API token is correct

## 💾 BACKUP & RECOVERY

**Automatic daily backup at 3 AM UTC**
- Location: `/docker/backups/twenty_YYYYMMDD.sql.gz`
- Retention: 30 days

```bash
ssh root@213.155.28.121 << 'BASH'
```
